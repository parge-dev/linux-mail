# LinuxMail 📬

A simple terminal email client for Linux using IMAP + SMTP.

## Features
- View inbox emails
- Send emails
- Supports Gmail and custom providers
- CLI-based lightweight mail tool

## Requirements
- a email
- a app password
- python

## Install

```bash
git clone https://github.com/parge-dev/linuxmail.git
cd linuxmail
chmod +x linuxmail.py
python3 linuxmail.py
